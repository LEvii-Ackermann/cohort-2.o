**Local Storage** 


**1.** localStorage.cleat() - to clean local storage

**2.** localStorage.setItem("key","value") - to add new item or overwrite existing item 

**3.** localStorage.getItem("key") - to get any item

**4.** localStorage.removeItem("key") - to remove any item


session storage is the storage which removes when the tab is closed 
it is used like-
    sessionStorage.setItem("user", "Sarthak")
